getUserProfile

This method retrieves information about the current user. It communicates with the server to fetch the user details.

Function Signature

const userInfo = await getUserProfile();

Example


const userInfo = await getUserProfile();
if (userInfo.code === 0) {
   //success
} else {
  // fail
}

Returns

This function returns a Promise which resolves to an object containing:

  • code: A number representing the result of the operation (0 for success, other values indicate failure).
  • data: An object containing the user information if the operation is successful.
  • msg: A string describing the outcome of the operation, especially in case of errors.

Error Handling

If the retrieval fails, the code will be non-200, and the message will provide details about the error, such as server-related issues or network errors.

Important Note

Ensure that the server is reachable before calling this method to avoid unnecessary errors.
