createFolder

This method allows you to create a group by specifying its name and parent ID. It validates the provided inputs and communicates with the server to perform the creation process.

Function Signature

const result = await createFolder(name, parent);

This method sends a request to create a new group under a specified parent group.

Example

const result = await createFolder('New Group', 10);
if (result.code === 0) {
    console.log(result.data); // Success
} else {
    console.error(result.message); // Failure
}

Parameters

  • name: String - The name of the group. This is a required parameter and must be a valid string.
  • parent: Number - The ID of the parent group. This parameter is not mandatory and creates a root node file group by default. When the input value is not empty, it must be a valid numeric ID.

Returns

This function returns a Promise which resolves to an object containing:

  • code: A number representing the result of the operation (0 for success, other values indicate failure).
  • data: The newly created group object if the operation is successful.
  • msg: A string describing the outcome of the operation, especially in case of errors.

Error Handling

If the group creation fails, the code will be non-200, and the message will provide details about the error, such as invalid group name, invalid parent ID, or server-related issues.

Important Note

Ensure that both name and parent parameters are valid before calling the method to avoid unnecessary errors. The method uses validators to check the input values, and it will return any validation errors if the inputs are invalid.
