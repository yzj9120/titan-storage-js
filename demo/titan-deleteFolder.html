deleteFolder

This method allows you to delete a group by specifying its group ID. It sends a request to remove the group from the server.

Function Signature

const result = await deleteFolder({ groupId });

This method communicates with the server to delete the specified group.

Example

const result = await deleteFolder({ groupId: 1001 });
if (result.code === 0) {
    console.log(result.data); // Success
} else {
    console.error(result.message); // Failure
}

Parameters

  • groupId: Number - The ID of the group to delete. This is a required parameter and must be a valid group ID.

Returns

This function returns a Promise which resolves to an object containing:

  • code: A number representing the result of the operation (0 for success, other values indicate failure).
  • data: The status of the delete operation if it is successful.
  • msg: A string describing the outcome of the operation, especially in case of errors.

Error Handling

If the deletion operation fails, the code will be non-200, and the message will provide details about the error, such as an invalid group ID or server-related issues.

Important Note

Ensure that the groupId parameter is valid before calling the method to avoid unnecessary errors.
