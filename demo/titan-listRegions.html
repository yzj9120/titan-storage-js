listRegions

This function is used to obtain a list of node regions and return the data as a Promise. .

Function Signature

Promise<Object> listRegions()

Example

TitanStorage.getInstance().listRegions().then(data => console.log(data));

Parameters

This function does not accept any parameters.

Returns

The returned Promise resolves to an object with the following structure:


{
  "code": 0,
  "data": {
    "area_maps": [
      { "key": "China", "value": "China" },
      { "key": "HongKong","value": "HongKong" }
    ],
    "list": [
      "China",
      "HongKong"
    ]
  }
}

Data Structure

The returned object contains the following fields:

  • code: Status code of the request, 0 indicates success.
  • msg: Message describing the result, returned when the status code is not 0.
  • data: An object that contains two fields:
    • area_maps: An array of objects representing key-value pairs for each area, for example 
      { "key": "China", "value": "China" }
    • list: An array containing the names of all areas, for example 
      ["China", "UnitedStates", "Vietnam", ...]

Error Handling

If an error occurs during the fetching process, the code will be non-zero, and the msg will provide details about the error.

Important Note

Always check the status of the response to handle different outcomes appropriately.
