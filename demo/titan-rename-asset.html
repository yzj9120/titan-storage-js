renameAsset

  • renameAsset(options = { assetId: -1, name: '' }): Promise<Object>

  • 该函数用于重命名指定的文件。用户必须提供文件 CID 和新的名称。

    Example
    TitanStorage.renameAsset({
  assetId: 12345,
  name: 'Renamed Asset'
}).then(data => {
  console.log('Asset renamed successfully:', data);
}).catch(error => {
  console.error('Error renaming asset:', error);
});

    Parameters

    • assetId: number

      文件的 CID。必须提供。

    • name: string

      新文件名称。必须提供，不能为空或仅由空白字符组成。

    Returns Promise<Object>

    返回一个 Promise 对象，解析为包含重命名操作结果的响应数据。

    Success Response

    成功返回的数据结构如下：

    {
    "code": 0,
    "data": {
        "msg": "success"
    }
}

    Success Response Fields

    • code: number

      状态码，0 表示成功。

    • data: Object

      返回数据的主体对象。

      • msg: string

        成功信息。

    Failure Response

    失败返回的数据结构如下：

    {
    "code": -1,
    "err": 1004,
    "msg": "Server Busy"
}

    Failure Response Fields

    • code: number

      状态码，-1 表示失败。

    • err: number

      错误码，具体的错误类型。

    • msg: string

      错误消息，描述错误的具体原因。

    Throws

    如果名称为空或仅包含空白字符，或在重命名文件过程中发生错误，将抛出一个错误。
