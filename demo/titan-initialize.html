Initialize the Titan Storage SDK

To begin using the Titan Storage SDK, you first need to initialize it. This involves providing your token, URL, and setting the debug mode.

Function Signature

const result = TitanStorage.initialize({ token: 'your-token', url: 'your-url', debug: true });

This method initializes the SDK and ensures that it is only initialized once. It returns a result containing a status code and message.

Example

const result = TitanStorage.initialize({ token: 'your-token', url: 'your-url', debug: true });
if (result.code === 0) {
    console.log(result); // Success
} else {
    console.error(result); // Failure
}

Parameters

  • token: String - Your token for the SDK. This parameter is required for authentication and must not be empty.
  • url: String - The URL for the SDK endpoint. This parameter must be provided.
  • debug: Boolean - Whether to enable debug mode. The default value is true.

Returns

This function returns an object containing:

  • code: A number representing the result of the initialization (0 for success, other values indicate failure).
  • msg: A string describing the outcome of the initialization.

Error Handling

If the initialization fails, the code will be non-zero, and the msg will provide details about the error, such as an invalid token or if the SDK was already initialized.

Important Note

If the SDK is not initialized correctly, you may lose access to certain features. Always check the result of the initialization method.
